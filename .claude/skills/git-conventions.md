---
name: git-conventions
description: Git işlemleri yaparken, commit veya branch oluştururken devreye gir.
---

Git işlemlerinde:
- Branch: `feat/`, `fix/`, `chore/`, `docs/` prefix'leriyle başlat
- Commit: `<type>: <Türkçe açıklama>` formatı
- Tek commit = tek mantıksal değişiklik
- WIP commit'leri push etme
- .env ve secret içeren dosyaları asla commit etme
